## 🔧 技术实现

### 核心函数

```python
# 1. 域名验证
verify_domain(url) -> (bool, source_name)

# 2. 风险评分
calculate_risk_score(title) -> (score, matched_risks)

# 3. 质量验证
verify_news_quality(title, url, source) -> {
    'score': int,
    'is_verified': bool,
    'risks': list,
    'stock_code': str,
    'stock_name': str
}

# 4. 股票信息提取
extract_stock_code(title) -> str
extract_stock_name(title) -> str
```

### 过滤规则

**严格过滤**（直接排除）：
- ❌ 宏观经济新闻（GDP、CPI、央行等）
- ❌ 市场分析评论（板块、概念、涨停等）
- ❌ 机构观点研报（分析师、券商、评级等）
- ❌ 国际政治新闻（特朗普、伊朗、战争等）

**例外保留**：
- ✅ 包含具体公司名称
- ✅ 包含股票代码
- ✅ 包含高风险关键词

## 🔐 安全保障

### 防误报机制

1. **来源白名单** - 只信任官方认证媒体
2. **风险关键词验证** - 必须匹配风险库
3. **质量评分门槛** - 低于 60 分自动过滤
4. **人工审核建议** - 高风险新闻建议人工复核

### 防漏报机制

1. **关键词扩展** - 10 类风险，100+ 关键词
2. **同义词匹配** - 支持多种表述方式
3. **权重评分** - 不同风险不同权重
4. **低门槛通过** - 60 分即可通过（总分 100）
